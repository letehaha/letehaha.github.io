Чому саме ця тема?

Люди, які пов’язані своєю роботою з ІТ-сферою, дуже часто працюють через мережу Інтернет. Щоб розповісти про себе, вони створюють веб-сайти, на яких вони розповідають про свої навички, демонструють приклади робіт та представляють способи для зв’язку з ними. Це необхідно для успішної комерційної діяльності.

Оскільки не всі вони можуть власноруч створити веб-сайт, часто вони використовують готові шаблони. Саме тому було вирішено обрати тему дипломної роботи “Розробка шаблону для створення професіонального портфоліо”.

З чого почав?

Спочатку було визначено з типом сайту – сайт-візитка. Саме такий сайт ідеально підходить для представлення робіт програміста. Він є невеликим, зручним, надає можливість представити всю необхідну інформацію про особу та її роботи і надати можливість зворотнього зв’язку. Такий тип сайту легкий у підтримці та має можливості до маштабування.

Основними вимогами було визначено:

  • веб-сайт у вигляді Landing Page;
  • адаптивність та кроссбраузерність;
  • новітній дизайн;
  • покращена оптимізація.

У розробці допомагали

css-tricks.com – сервіс, інформація з якого допомогла продумати концепцією дизайну, правильно стилізувати шрифти та застосувати паттерни CSS3.

habrahabr.ru – ресурс для IT-спеціалістів, який містить безліч інформації по оптимізації пошукових запросів та швидкості завантаження веб-сайту.

jsraccoon.ru – невеличкий веб-сайт для js-розробників, на якому можна знайти нестандартні реалізації паттернів програмування та просто цікаві статті.

codepen.io – "пісочниця" для Front-End розробників, в якій представлена безліч цікавих робіт звичайних людей, з якими можна ознайомитися та почерпнути для себе щось нове.

github.com – один з найбільших веб-сервісів для спільної розробки програмного забезпечення. Величезна кількість різноманітних проктів з відкритим кодом надають змогу глянути на код інших розробників та зробити для себе певні висновки.

Висновки

У ході написання дипломної роботи було досліджено технології та інструментальні засоби для розробки шаблону професійного веб-сайту портфоліо.
